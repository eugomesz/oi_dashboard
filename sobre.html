Sobre o Projeto

Monitoramento

Este projeto tem como objetivo monitorar e analisar dados ambientais em tempo real, fornecendo insights valiosos sobre qualidade do ar, temperatura, umidade e outros parâmetros relevantes.

Tecnologias utilizadas: HTML5, CSS3, JavaScript, Bootstrap, ApexCharts

Objetivos:

  • Visualização clara de dados ambientais
  • Alertas em tempo real para anomalias
  • Histórico de medições para análise temporal
